Power BI - Publish to Web demo

This page is a demonstration of Power BI's Publish to Web feature, for embedding interactive (but unsecured) Power BI report content. Below, the content is embedded in an iframe, which can be sized to suit. At the bottom of the page is a link to open the content in a new tab/window.



Note the page navigation control is at bottom-centre, e.g. 1 of 8.
